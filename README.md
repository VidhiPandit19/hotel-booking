# 🏨 Hotel Booking System

The **Hotel Booking System** is a web-based application that allows users to search for hotels, view room details, and make reservations online.
This project demonstrates how a hotel reservation platform works by integrating a simple and user-friendly interface with booking functionality.

The system helps users easily browse available rooms, check availability, and manage their bookings efficiently.

---

## ✨ Features

* 🔍 Browse available hotels and rooms
* 📅 Select check-in and check-out dates
* 🛏 View room details and pricing
* 📝 Book rooms through a simple reservation form
* 📱 Responsive design for desktop and mobile
* ⚡ Fast and user-friendly interface

## ⚙️ Installation

Clone the repository:

```
git clone https://github.com/VidhiPandit19/hotel-booking.git
```

Navigate to the project folder:

```
cd hotel-booking
```

Install dependencies:

```
npm install
```

Run the development server:

```
npm start
```

Open the application in your browser:

```
http://localhost:3000
```
